# Минимальный шаблон (3 страницы)

Стек:
- Node.js + Express
- Sequelize + PostgreSQL
- EJS + Bootstrap 5
- express-session
- bcrypt

## Страницы
1. `/register` — регистрация
2. `/login` — авторизация
3. `/` — главная (текст задания билета), только после входа

## Запуск
```bash
cp .env.example .env
npm install
npm run dev
```
